# Temporary scripts and test UI

- **server.py** — Run from project root: `python temp/server.py`  
  Serves this folder and `/api/platforms.json` from `data/json/platforms.json`.  
  Open http://127.0.0.1:8765

- **index.html** — Simple test UI: list platforms, filter by category, search, click for detail.

Do not commit large or permanent assets here; this folder is for quick local testing.
